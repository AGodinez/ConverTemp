    ConverTemp

    ConverTemp es una aplicación conversora de unidades de temperatura para dispositivos con Firefox OS, tambien funciona en Firefox Desktop pero en la siguiente versión se mejorará para esta plataforma.


    Tecnologías usadas

    HTML5 - JQuery Mobile 1.4.4 - JQuery 1.11.1 - webL10n

    Autor y Contribuidores
